Additional Data

Here, you will find some additional data packages for SUMO.

Additional data for Version 0.9.4


  • SUMO-Frida 0.9.4
    The converted network from the city of Osnabrück from the Frida project.
    This data contain the converted network only. There is no demand that could be put onto it.
    It is just an example of converting networks from shape-files. Read more on the problems in Importing the Network of Osnabrück from the Frida-project.
    Download as: .zip [sumo-frida-built-0.9.4.zip, ~6.0MB] or .tar.gz [sumo-frida-built-0.9.4.tar.gz, ~5.7MB]

    You can also download the source together with the NETCONVERT-configurations and the additional types file which was described in Importing the Network of Osnabrück from the Frida-project.
    Download as: .zip [sumo-frida-src-0.9.4.zip, ~640kB] or .tar.gz [sumo-frida-src-0.9.4.tar.gz, ~620kB]
