Download Section

Here, you will find SUMO both as sources and as compiled binaries (for MS Windows only) and also the libraries needed to compile and run it. Please remark that we can not longer provide a direct download due to sourceforge's disc quotas.

As the program is still under development and is extended continuously, we advice you to use the latest sources from our svn. Normally, they should compile and be newer than the releases are.

Please contact us if you have any problems. You will find the needed information at the Contact-page.

Follow the link to http://www.gnu.org/licenses/gpl.html to see the Licence.

See ChangeLog for further information about the latest release.

Please note that since version 0.9.6 the "developer" branch is discontinued; we only have a "stable" branch, now.
