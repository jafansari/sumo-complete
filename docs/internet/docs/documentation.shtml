Documentation

Most of the documentation was done using the DocBook format. You may either browse it online or download it as a tarball of html-pages or as pdf-documents.

We apologize that it still is in progress and is incomplete...

The following documentation is available:

  • User Documentation
    A "complete" documentation of how to use SUMO and its applications. Available as:
    On-Line Manual (HTML), split by chapters
    On-Line Manual (HTML), one file
    Manual (PDF)

  • More On...
    Some additional information on special subjects (traffic lights, ...).

  • Some relevant or assigned publications
    Some publications you may refer to.

  • Frequently Questions and Answers (FAQ)
    Some other things that have not found their way into the documentation.
    You may also take also a look into our Wiki and into our Mailing Lists.
