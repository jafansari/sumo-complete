Download Section

Here, you will find SUMO both as sources and as compiled binaries (for MS Windows and Linux only) and also the libraries needed to compile and run it.

As the program is still under development and is extended continuously, we advice you to use the latest sources from our cvs. Normally, they should compile and be in a better state than the releases are.

Please contact us if you have any problems. You will find the needed information at the Contact-page.

Follow the link to http://www.gnu.org/licenses/gpl.html to see the Licence.

See Changes for further information on the changes within the latest releases. See Release Policy for information on the meaning of the release names.

SUMO - Latest Stable Release (Version 0.8.2.2)


"stable"; All input and command line options will not be changed within the 0.8.2.x-line. Use this for your projects if you want to use the same settings for a longer time.

Included: sumo, netconvert, netgen, jtrrouter, duarouter, od2trips.

Please report bugs to or at http://sourceforge.net/tracker/?group_id=45607&atid=443421.
Please remind that several option names have changed! The most important: "net-files" (router, simulation) is now "net-file"...
Check user documentation and man-pages for this.

    Contains the binaries, all dlls needed and some examples.
    Download as: [sumo-winbin-0.8.2.2.zip, ~2.6MB]

  • The last stable Linux binaries.
    Please use the source distribution; A compiled version is not supplied.

  • The last stable source-tarball. This one was built on 14.07.2005. Includes sources, examples, Linux-Makefiles and MSVC++ 6.0 and MS Visual Studio .net-project workspace.
    Download as: [sumo-src-0.8.2.2.tar.gz, ~3MB] [sumo-src-0.8.2.2.zip, ~4.2MB]
