Download Section

Here, you will find SUMO both as sources and as compiled binaries (for MS Windows and Linux only) and also the libraries needed to compile and run it.

As the program is still under development and is extended continuously, we advice you to use the latest sources from our cvs. Normally, they should compile and be in a better state than the releases are.

Please contact us if you have any problems. You will find the needed information at the Contact-page.

Follow the link to http://www.gnu.org/licenses/gpl.html to see the Licence.

See Changes for further information on the changes within the latest releases. See Release Policy for information on the meaning of the release names.
